About

This is a sample To Do application built with Angular 17. It was created for an interview technical assessment at Xello.

Project Overview

The app uses the upcoming Angular 17 (beta) and demonstrates core Angular capabilities:

  • Components
  • Services
  • Routing
  • Pipes
  • Directives
  • Testing

Key Features

  • Add tasks with title, description, due date etc.
  • Edit and delete existing tasks
  • Search/filter tasks
  • Sorted by due date or priority
  • Responsive design

Built With

  • Angular
  • TypeScript
  • HTML
  • CSS

State Management

Task data is stored locally using browser local storage. May integrate state management library like NgRx in future.

Testing

Jest used for unit testing of components and services.

Installation

To install:

  1. Clone the repository
  2. Run npm install
  3. Run ng serve for a dev server
  4. Navigate to http://localhost:4200
